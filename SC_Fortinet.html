3.Address:		Tiene la siguiente estructura donde todos los datos son necesarios.
		
	

      
        

          id1
		  id2
		  id3
		  id4
		  id5
		  id6  
        

      
      
        

          edit
          name-address
          set type ipmask
		  set subnet 
		  1.2.3.4/32
		  next
        

        

          edit
          name-address-fqdn
          set type fqdn
		  set fqdn 
		  my.address.com
		  next
        
       
      
    

	
4. Perfil WEB:	Tiene la siguiente estructura donde todos los datos son necesarios. 
		Solo son necesarios las subcategorias con accion bloquear y monitor.
	


      
        

          id1
		  id2
		  id3
		  id4
		  id5
		  id6  
        

      
      
        

          edit 0
          set category
          26
		  set action
		  block
		  next
        

         

          edit 0
          set category
          49
		  set action
		  monitor
		  next
        
   
      
    



5.  URL Filter:	Tiene la siguiente estructura donde todos los datos son necesarios. 



      
        

          name
		  type
		  action
        

      
      
        

          *.url.com.co
          wildcard
          exempt
        

         

          www.url.com.co
          simple
          allow
        
 
			

          *xxx*
          regex
          block
